Card Layout Script
A language for designing cards

Extract the contents of this folder somewhere handy, like into Documents/CLSRenderer then you can run the app directly, make shortcuts, or right click a .cls file and use open with to associate the app with the file type
See codlark.itch.io/cls for more information
